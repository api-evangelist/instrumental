# Instrumental (instrumental)
Instrumental is a platform for manufacturing intelligence that helps companies detect and prevent product failures during the manufacturing process. The Instrumental API enables programmatic access to retrieve unit and inspection test data and metadata from Instrumental projects.

## APIs

### Instrumental Data Read API
The Instrumental Data Read API provides programmatic access to retrieve unit and inspection test data and metadata from Instrumental projects. It enables software to access manufacturing data for analysis and integration with other systems.
